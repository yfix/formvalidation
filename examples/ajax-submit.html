Using Ajax to submit the form

By triggering the success.form.bv event, we can use Ajax to submit the form data.

{% highlight javascript hl_lines="6 8" %} $(document).ready(function() { $(form) .bootstrapValidator({ ... options ... }) .on('success.form.bv', function(e) { // Prevent form submission e.preventDefault(); // Get the form instance var $form = $(e.target); // Get the BootstrapValidator instance var bv = $form.data('bootstrapValidator'); // Use Ajax to submit form data $.post($form.attr('action'), $form.serialize(), function(result) { // ... Process the result ... }, 'json'); }); }); {% endhighlight %}

In the example above, you can see the form is prevented from doing default submission by using

{% highlight javascript %} e.preventDefault(); {% endhighlight %}
